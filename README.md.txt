# ⚙️ Parametri Frese

Applicazione web per salvare e gestire i parametri delle frese tramite LocalStorage.

## Funzionalità
- Inserimento nuovo record
- Modifica dei parametri
- Eliminazione
- Ricerca dinamica
- Esportazione CSV (se implementata)
- Interfaccia moderna e responsive

## Tecnologie
- HTML
- CSS
- JavaScript
- LocalStorage

## Struttura del progetto
- index.html → interfaccia principale
- style.css → stile grafico
- app.js → logica dell'app
- manifest.json → supporto PWA
- icons/ → icone app

## Installazione
Non serve installazione: basta aprire `index.html` nel browser.
